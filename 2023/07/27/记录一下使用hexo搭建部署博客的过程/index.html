创建个人仓库

取名方式：用户名.github.io(以后的访问域名)
只有这样，将来要部署到GitHub page的时候，才会被识别

安装hexo

全局安装hexo命令

初始化hexo

*在准备要存放博客的文件路径下打开cmd *

创建成功，启动服务


生成SSH添加到GitHub

设置全局用户名和邮箱

查看设置的全局用户名和游戏

创建SSH,一路回车

生成成功，根据它提示的地址找到生成的.ssh文件夹

id_rsa 私钥

id_rsa.pub 公钥

将公钥放到GitHub

公钥放在GitHub上，这样当你连接GitHub自己的账户时，它就会根据公钥匹配你的私钥，当能够相互匹配时，才能够顺利的通过git上传你的文件到GitHub上了。

  1. setting 中找到 SSH and GPG keys
  2. new SSH Key 创建一个新的ssh key
  3. 将公钥—id_rsa.pub文件中的内容复制到key的输入框中
  4. 点击添加即可

验证ssh是否成功

hexo部署到GitHub

修改_config.yml文件，拉到最下面

repo，就是git仓库地址
安装部署的命令 deploy-git，当前文件下安装

进行部署

hexo clean 清除了你之前生成的东西
hexo generate 生成静态文章，可以用 hexo g 缩写
hexo deploy 部署文章，可以用 hexo d 缩写
