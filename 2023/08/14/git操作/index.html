合并分支单条

如果需要将A分支的某一条代码合并到B分支

首先要切换到B分支上，然后选择A分支上需要合并的那条代码的修订号

xxx 为修订号

如果遇到了冲突,那就先解决冲突,然后执行 git cherry-pick –continue 继续应用剩余的提交

如果您想跳过当前的提交，可以使用git cherry-pick –skip命令来跳过这个提交，直接应用下一个提交

如果您不想继续当前的cherry-pick操作，可以使用git cherry-pick –abort来取消整个操作，这将回到操作开始前的状态。
